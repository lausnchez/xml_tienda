# Prácticas de tienda de XML
>[!IMPORTANT]
> Práctica importante para el examen, se tomará como ejercicio de preparación
1. Menú
2. Agregar Compra
3. Agregar Venta
4. Mostrar datos

## Formato final del XML
```
<datos>
    <ventas>
        <producto codigo="1" tipo="A">
            <nombre></nombre>
            <descripcion></descripcion>
            <precio></precio>
        </producto>
    </ventas>
    <compras>
        <producto codigo="1" tipo="A">
            <nombre></nombre>
            <descripcion></descripcion>
            <precio></precio>
        </producto>
    </compras>
</datos>
```
